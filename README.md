# Sitio Web - Especialización en Ingeniería de Software UAI

## Descripción

Este sitio web educativo contiene el contenido académico completo de la Especialización en Ingeniería de Software de la Universidad Abierta Interamericana (UAI), organizado en 16 clases comprehensivas de aproximadamente 2000 palabras cada una.

## Estructura del Sitio

- **index.html**: Página principal con navegación
- **css/styles.css**: Estilos del sitio
- **js/script.js**: Funcionalidad de navegación dinámica
- **clases/**: Directorio con las 16 clases en HTML

## Módulos del Curso

### Módulo I: Fundamentos
1. Introducción a la Especialización en Ingeniería de Software
2. Métodos Formales en la Ingeniería de Software
3. Ingeniería de Requerimientos
4. Diseño y Arquitectura de Software

### Módulo II: Tecnologías Avanzadas
5. Elementos Avanzados de Bases de Datos
6. Verificación Formal de Software
7. Diseño Avanzado Orientado a Objetos
8. Reingeniería y Refactoring

### Módulo III: Desarrollo de Aplicaciones
9. Desarrollo de Aplicaciones Empresariales
10. Ingeniería Web Dirigida por Modelos
11. Metodologías Ágiles
12. Desarrollo de Aplicaciones Móviles

### Módulo IV: Calidad y Práctica Profesional
13. Usabilidad y Experiencia de Usuario
14. Auditoría Informática y Calidad de Software
15. Práctica Profesional
16. Trabajo Final de Especialización

## Características

- Diseño responsive y profesional
- Navegación lateral con todos los capítulos
- Diagramas Mermaid integrados para visualización
- Contenido académico riguroso de 2000 palabras por clase
- Interfaz limpia y fácil de navegar

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript ES6
- Mermaid.js para diagramas

## Cómo Usar

1. Abrir `index.html` en un navegador web moderno
2. Navegar entre las clases usando la barra lateral
3. Los diagramas Mermaid se renderizarán automáticamente
