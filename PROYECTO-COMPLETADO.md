# Proyecto Completado: Sitio Web Educativo - Diplomado en Ingeniería de Software

## Resumen del Proyecto

Se ha creado exitosamente un sitio web educativo completo con contenido académico para el Diplomado en Ingeniería de Software de Liskov Ed Tech.

## Características del Sitio

### Estructura
- **Header profesional** con información del diplomado
- **Barra lateral de navegación** con 16 clases organizadas en 4 módulos
- **Contenido principal** con carga dinámica de clases
- **Footer** con información institucional

### Contenido Académico
- **16 clases completas** de aproximadamente 1500-2500 palabras cada una
- **Tono académico riguroso** con fundamentos teóricos
- **6 subtemas principales** desarrollados en cada clase
- **Diagramas Mermaid** integrados para visualización de conceptos
- **Navegación entre clases** con botones anterior/siguiente

### Módulos del Curso

#### Módulo I: Fundamentos
1. Introducción al Diplomado en Ingeniería de Software
2. Métodos Formales en la Ingeniería de Software
3. Ingeniería de Requerimientos
4. Diseño y Arquitectura de Software

#### Módulo II: Tecnologías Avanzadas
5. Elementos Avanzados de Bases de Datos
6. Verificación Formal de Software
7. Diseño Avanzado Orientado a Objetos
8. Reingeniería y Refactoring de Software

#### Módulo III: Desarrollo de Aplicaciones
9. Desarrollo de Aplicaciones Empresariales
10. Ingeniería Web Dirigida por Modelos
11. Metodologías Ágiles
12. Desarrollo de Aplicaciones Móviles

#### Módulo IV: Calidad y Práctica Profesional
13. Usabilidad y Experiencia de Usuario
14. Auditoría Informática y Calidad de Software
15. Práctica Profesional
16. Trabajo Final de Diplomado

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Diseño responsive y profesional con variables CSS
- **JavaScript ES6**: Navegación dinámica y carga de contenido
- **Mermaid.js**: Generación de diagramas técnicos

## Cómo Usar el Sitio

1. Abrir `index.html` en un navegador web moderno
2. Usar la barra lateral para navegar entre las clases
3. Los diagramas Mermaid se renderizarán automáticamente
4. Usar los botones "Anterior" y "Siguiente" para navegación secuencial
